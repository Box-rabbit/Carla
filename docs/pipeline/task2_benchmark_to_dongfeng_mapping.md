# 任务2：东风三类场景与现有 Benchmark 场景映射

## 总体映射
| 东风场景类别 | 东风要求关键词 | 可参考 benchmark | 第一版实现建议 |
|---|---|---|---|
| 基础操控 | 启动/停止、加速/减速、左转/右转、变道 | LMDrive/LangAuto, Leaderboard | 直道速度控制、变道、路口转向 |
| 复杂避障 | 行人、公交站、慢车、锥桶、施工绕行 | ScenarioRunner, Leaderboard, Bench2Drive | 行人横穿、锥桶绕行、慢车超越 |
| 应急响应 | 雨夜、低光、突发加塞、危险路况、紧急制动 | ScenarioRunner, Bench2Drive, Leaderboard | 突发加塞、雨夜减速、施工并道 |

## 基础操控
| 场景ID | 任务 | 成功判定 |
|---|---|---|
| B01 | 保持车道 | 未偏离车道，路线完成，无碰撞 |
| B02 | 目标速度控制 | 规定时间内达到目标速度 ±5 km/h |
| B03 | 左/右变道 | 进入目标车道且保持稳定 |
| B04 | 路口左/右转 | 通过指定路口并进入目标道路 |
| B05 | 停车 | 目标区域内速度接近 0，无碰撞 |

## 复杂避障
| 场景ID | 任务 | 成功判定 |
|---|---|---|
| C01 | 行人横穿减速 | 与行人保持安全距离，无碰撞，速度下降 |
| C02 | 慢车超越 | 安全变道并超过慢车 |
| C03 | 锥桶绕行 | 不碰撞锥桶，完成绕行 |
| C04 | 公交站减速 | 到达公交站区域前减速至目标速度 |
| C05 | 施工绕行并回归车道 | 绕行并返回目标车道 |

## 应急响应
| 场景ID | 任务 | 成功判定 |
|---|---|---|
| E01 | 突发加塞紧急制动 | 未碰撞，制动及时，TTC 风险下降 |
| E02 | 前车急刹 | 未追尾，制动距离合理 |
| E03 | 雨夜危险路况减速 | 速度低于安全阈值，无碰撞 |
| E04 | 施工路段减速并道 | 施工区前完成减速和并道 |
| E05 | 临时行人横穿 | 未碰撞，响应延时达标 |
